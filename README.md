# Knowledge Nexus — 跨领域知识关联引擎

> 借助 AI 发现看似无关事物之间的深层关联，将孤立的知识点编织成互联互通的知识网络。

## 🎯 项目愿景

人类知识分散在不同领域中，但本质上许多概念跨域相通：
- 生物学的 **自然选择** ↔ 计算机的 **遗传算法**
- 物理学的 **退火过程** ↔ 优化领域的 **模拟退火**
- 神经科学的 **神经网络** ↔ 深度学习的 **人工神经网络**
- 经济学的 **博弈论** ↔ 多智能体的 **强化学习**

**Knowledge Nexus** 旨在：
1. **构建领域内知识图谱** — 梳理单个领域（如计算机科学）中各 SOTA 工作之间的引用、继承、改进关系
2. **发现跨领域关联** — 利用 AI 识别不同领域知识之间的结构相似性与概念迁移
3. **生成新知识假说** — 基于已有关联模式，推断潜在的跨域启发与创新方向

## ✨ 核心功能

### 📚 知识库管理
- 论文/工作条目录入（PDF、标题、摘要、关键贡献、领域标签）
- 自动解析 PDF 提取元数据（标题、作者、摘要、参考文献）
- 支持批量导入（BibTeX、arXiv ID、DOI）

### 🕸️ 知识图谱可视化
- 交互式图谱展示节点（论文/概念）与边（关联关系）
- 支持按领域、时间线、影响力等维度筛选
- 缩放、拖拽、聚焦子图等交互操作
- 关联类型标注（引用、改进、启发、类比、对比等）

### 🤖 AI 关联发现
- 基于 LLM 分析论文内容，自动发现领域内关联
- 跨领域语义相似度计算，发现概念迁移
- 生成关联假说并附带推理解释
- 支持用户确认/否决 AI 建议，持续优化

### 🕷️ 智能论文爬取
- 多数据源自动爬取（Semantic Scholar、arXiv、DBLP、Papers with Code、OpenAlex）
- 按引用量、顶会/顶刊、SOTA 记录等综合评分筛选开创性工作
- 自动下载 Open Access 论文 PDF
- 断点续爬、去重、限流，礼貌且可靠
- 一键爬取预设方案（如：计算机科学近 10 年里程碑论文）

### 🔍 智能检索
- 全文搜索 + 语义搜索
- "给我找与 X 概念最相关的其他领域工作"
- 知识路径查询（从概念 A 到概念 B 的关联路径）
